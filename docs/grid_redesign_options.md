# 📐 이요하우스 격자선 변환 모션 개선안 (Redesign Options)

이요하우스의 그리드 정체성을 유지하면서도, 페이지 전환 시 격자선들이 너무 복잡하게 얽히고 춤추어 눈이 피로했던 문제를 해결하기 위해 **가장 유력한 2가지 대안(소프트 슬라이드 & 라인 페이드)**의 설계 및 구현 방식을 상세히 정리합니다.

---

## 📌 Option 1 (2안): 단일 세로 구분선 소프트 슬라이드 (Single Axis Soft-Slide)

> **핵심 개념**: 4개의 화면 가장자리 액자 격자선(상/하/좌/우 각 1줄)은 모든 상황에서 완전히 고정됩니다. 공간 분할을 가르는 **단 1개의 세로 격자선(보조선)만** 페이지 성격에 맞춰 조용히 좌우로 슬라이드 이동합니다.

### 1. 동작 메커니즘
* **격자 고정**: 상단 가로선(`.grid-row-1`), 하단 가로선(`.grid-row-2`), 좌측 세로선(`.v-line`의 line1)은 뷰포트 가장자리에 액자 형태로 완벽히 고정됩니다.
* **동적 이동**: 공간을 분할하는 세로 격자선 딱 1줄(`.v-line`의 line3)만 `useGridLayout.ts`에서 계산된 프리셋 좌표에 따라 좌우로 부드럽게 미끄러집니다.
* **교차 마커 동기화**: 세로선 위에 얹어진 마커(`.top-v-3` 등)도 해당 세로선의 `left` 좌표와 100% 동일하게 묶여 함께 부드럽게 슥 이동합니다.

### 2. 프리셋별 예시 흐름
1. **메인 페이지 (`main`)**: 세로 보조선이 화면 우측 끝 구석(`calc(var(--stage-width) - var(--unit))`)에 차분히 대기하여 넓은 화면 공간을 확보합니다.
2. **워크숍 페이지 (`workshop`)**: 세로 보조선이 왼쪽(예: `60%` 지점)으로 슥 부드럽게 밀려와 넓은 워크숍 상세 카드 영역과 우측의 서브 공간 경계를 시각적으로 재단합니다.
3. **다이어리 페이지 (`diary`)**: 세로 보조선이 우측(예: `var(--unit)`) 부근으로 이동하여 캘린더 공간의 우측 벽면 역할을 정교하게 수행합니다.

### 3. 장단점 분석
* **👍 장점**: 
  - 여러 개의 선과 마커가 동시에 오르내리지 않고 단 하나의 세로축만 움직여 시각적 피로감이 80% 이상 격감합니다.
  - 이요하우스 고유의 "격자선이 움직여 공간을 연출하는 디자인 위트"를 정갈하게 유지할 수 있습니다.
* **👎 단점**: 
  - 세로선이 이동할 때 콘텐츠 영역의 크기가 급격히 바뀔 수 있어, 셀 영역 내 콘텐츠의 부드러운 페이드 전환 타이밍이 맞아야 합니다.

## 📌 Option 2 (3안): 움직임 없는 격자선 페이드 (No-Movement Line Fade)

> **핵심 개념**: 격자선들이 물리적인 위치(CSS `left`, `top`) 이동을 완전히 멈춥니다. 격자선들은 각 프리셋별 완벽한 배치 좌표에 굳건히 고정되어 있으며, 페이지 전환 시 필요한 선들만 **불빛이 켜지고 꺼지듯이 부드럽게 페이드인/아웃**됩니다.

### 1. 동작 메커니즘
* **정적 위치 고정**: 모든 격자선은 transition을 통해 물리적으로 움직이지 않고, `left`와 `top` 값이 즉시(instant) 바뀝니다.
* **투명도 트랜지션**: 선과 마커들의 스타일 규칙에 `transition: opacity 0.3s ease, visibility 0.3s;`만 부여합니다.
* **전환 연출**: 버튼을 누르면 이전 페이지용 격자선과 마커들이 0.3초간 스르륵 사라지고(opacity 0), 새로운 페이지용 격자선과 마커들이 제자리에서 스르륵 나타납니다(opacity 1).

### 2. 장단점 분석
* **👍 장점**:
  - 화면이 덜컹거리거나 선이 찢어지는(Tearing) 현상이 물리적으로 원천 차단됩니다.
  - 모바일과 저사양 브라우저 환경에서 브라우저 렌더링 부하가 가장 적고 안정적입니다.
  - 갤러리 미술관 사이트와 같은 극단적인 정적 미니멀리즘을 선사합니다.
* **👎 단점**:
  - 격자선이 부드럽게 미끄러지며 공간을 가르는 이요하우스 고유의 동적인 맛과 인터랙티브한 재미가 다소 반감됩니다.

## 🛠️ 추천 및 최종 선택 (Option 1 실행 전략)

피드백해주신 대로 **2안(단일 세로선 소프트 슬라이드)**이 이요하우스의 감성과 미니멀함을 동시에 챙길 수 있는 최선의 선택입니다. 이를 실현하기 위해 다음과 같이 코드를 점검 및 정돈할 예정입니다.

1. **`useGridLayout.ts` 변수 정리**: 가로선은 100% 고정값으로 두고, `--grid-line-x-3`과 같은 단일 수평 좌표만 transition이 동작하는 유일한 물리 축으로 설계합니다.
2. **콘텐츠 페이드 타이밍 싱크**: 세로 격자선이 슬라이드 이동하는 시간(약 440ms~660ms) 동안, 내부 콘텐츠는 먼저 불투명도 0으로 빠르게 꺼졌다가 격자선이 목적지에 도달하는 시점에 맞춰 부드럽게 켜지도록 연출을 다듬습니다.
